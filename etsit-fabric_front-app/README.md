# Etsit Front

Etsit front project
    
### Local Execution
```bash
git clone https://github.com/KairosDS/etsit-fabric/tree/master/etsit-fabric_front-app
npm install
npm start
```

- Open browser [here](http://localhost:8000/)

For implementation details consult the [docs for hapi](https://hapijs.com/api)
